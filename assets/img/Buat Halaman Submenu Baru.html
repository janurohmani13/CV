  1. Persiapkan File HTML: Pastikan Anda memiliki file HTML yang ingin Anda unggah ke GitHub. Anda bisa membuatnya menggunakan teks editor apa pun, seperti Visual Studio Code, Sublime Text, atau teks editor bawaan MacBook.

    • Buka GitHub di browser dan masuk ke akun Anda.
    • Di sudut kanan atas, klik tombol "+" dan pilih "New repository".
    • Beri nama repositori Anda, tambahkan deskripsi opsional, dan pilih opsi untuk membuat repositori publik atau pribadi.
    • Klik tombol "Create repository" untuk membuat repositori baru.

  3. Inisialisasi Repositori Lokal:

    • Buka Terminal di MacBook Anda.
    • Pindah ke direktori tempat Anda menyimpan file HTML dengan perintah cd path/to/your/directory.

  4. Inisialisasi Repositori Git Lokal:

    • Jalankan perintah git init untuk menginisialisasi repositori Git lokal di direktori tersebut.

  5. Tambahkan File HTML ke Repositori:

    • Jalankan perintah git add your-file.html untuk menambahkan file HTML ke daftar yang akan di-commit.
